When You Set The Width and Height Properties of an Element in CSS. You just set the Width and Height of the Content Area. To Calculate the Full Size of an Element, You must also add margin, padding and borders.
Total Width of An Element: - Width + Left Padding + Right Padding + Left Border + Right Border + Left Margin + Right Margin
Total Height of An Element: - Height + Top Padding + Bottom Padding + Top Border + Bottom Border + Top Margin + BottomMargin
