CSS stands for Cascading Style Sheets, used to style and visually format HTML documents.
It controls colors, fonts, spacing, layout, and animations on webpages.
The "cascading" mechanism means styles flow down and can be overridden based on specificity and source order.
CSS uses selectors to target HTML elements and apply rules using property-value pairs (e.g., color: red).
Styles can be written inline, internally (in a <style> tag), or externally (in a .css file).
CSS supports responsive design through media queries, adapting layouts to different screen sizes.
Modern CSS includes powerful tools like Flexbox, Grid, and custom properties (variables) for complex layouts.
It separates content (HTML) from presentation (CSS), making code cleaner and easier to maintain.
